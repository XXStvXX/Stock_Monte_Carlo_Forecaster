Stock Monte Carlo Forecaster
GitHub Model notes CLI

Monte Carlo risk lab for market scenarios.

A Python and Streamlit project that simulates stock price paths, estimates downside risk, and turns uncertainty into readable probability metrics.

View repository Deploy on Streamlit 
pip install -e ".[dev]"
streamlit run app.py

stock-forecast VFV.TO \
  --lookback-days 1095 \
  --horizon-days 252 \
  --paths 2000 \
  --annual-macro-bias 0.02
GBM engine

Reusable simulation module with reproducible seeds and positive log-return price paths.

Risk metrics

VaR, CVaR, probability of loss, target-hit probability, and simulation intervals.

Interactive app

Streamlit dashboard with ticker selection, macro bias, charts, and data inspection.

Engineering polish

Package metadata, CLI, unit tests, CI workflow, model notes, and example config.
